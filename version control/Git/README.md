# git-notes
|command|description|
|-|-|
|`git clone <repo-resource>`|clone a repo|
|`git init`|initialize a git repo|
|`git add [file(s)]`|stage the file(s)|
|`git checkout -b <branch-name>`|create the branch if it doesn't exists, and jump to the branch|
